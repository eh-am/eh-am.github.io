Setting up evdi after updating kernel to 5.15.12

2022/01/09 4-minute read
blog • technical • linux

intro

as you may know, I use an old ipad as a second screen. in a nutshell, it relies on the evdi kernel module to create a virtual display which is then accessed by the ipad via a VNC Viewer.

pne quirk is that almost every time I update the kernel, evdi (and consequently my ipad setup) also breaks. I am not super familiar with kernel shenanigans so I don’t know if it’s a fundamental issue or not. 1

that being said, fixing it is relative fine as long as:

  • a) there’s been a fix released
  • b) I remember the steps on how to reinstall the module

for a), well, running fedora helps a bit with this (I reckon it would be much worse with a rolling release distro, like arch). It’s still possible to mess it up though, so I usually wait a few weeks before updating the OS version. Also watching new releases help.

now for b), that’s what this blog post is for. I usually maintain notes for almost everything technical I do, but it’s designed to be a write-intensive workflow (which means I suffer between greping for the right keywords and sequentially going over my daily notes).

1
